9. Alerts / Banners

Dùng inline trong form để cảnh báo risk params.

⚠
Không có Stop-loss
Bot có thể mất toàn bộ vốn khi giá đi ngược. Khuyến nghị bật stop-loss tối thiểu −2%.
⚡
Leverage cao (10x)
Rủi ro thanh lý tăng mạnh. Xem lại position size và stop-loss.
✓
Config hợp lệ
Sẵn sàng gửi lên server.
ⓘ
Testnet mode
Bot đang chạy trên testnet — không sử dụng tiền thật.

10. Modal

Confirmation cho hành động nguy hiểm (Stop, Delete).

⚠ Stop bot "BTC Scalper v2"?
Bot đang có 3 vị thế mở với tổng giá trị $4,521.80. Dừng bot sẽ không đóng các vị thế — bạn cần đóng thủ công.

11. Toasts

✓
Config đã lưu
File JSON đã gửi tới server thành công.
✕
Không kết nối được exchange
Kiểm tra lại API key và IP whitelist.
⚠
Leverage vượt ngưỡng an toàn
Giá trị 20x có thể dẫn đến thanh lý.
ⓘ
Đang đồng bộ orders
Dự kiến hoàn thành sau 10s.
