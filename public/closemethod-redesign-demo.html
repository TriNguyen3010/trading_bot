So sánh với hiện tại: Tab Indicator luôn hiện song song Manual/TP-SL/ROI → 4 tab cùng lúc gây overwhelm cho novice. Bản này: 3 tab default (Manual / TP-SL / ROI — cover ~70% case), Advanced toggle inline gọn với title Close method để bật tab Indicator ở cuối list (cho power user). Mở tab Indicator → đã có sẵn 1 default group + 1 rule (candle.close > 0) — user edit thẳng, không phải click thêm. Backend & code Indicator exit giữ nguyên — chỉ thêm 1 layout-pref flag (showAdvancedClose) ở store để remember toggle state, và pass defaultGroupOnMount cho ConditionTreeBuilder (giống Entry).
