Grouping 12 fields into sections — Scheme D selected

Scheme D ✓ is the refined version of B with three tweaks: (1) "Phase 1" + "Bot basics" on a single header row, (2) each section collapsible using the same pattern as Phase 2's StrategySection, (3) merge Mode & leverage + Capital & positions → one Capital section. Schemes A/B/C kept below as reference.

Field inventory (12 fields)
  1. Bot name (required)
  2. Pair (required, datalist)
  3. Timeframe (required)
  4. Trading mode (Dry-run / Live · confirm dialog)
  5. Leverage (slider + number)
  6. Exchange (Hyperliquid, locked chip)
  7. Market type (Futures, locked chip)
  8. Margin mode (Cross / Isolated)
  9. Max open trades
  10. Stake currency
  11. Stake amount
  12. Dry-run wallet (only when tradingMode === 'dry-run')
